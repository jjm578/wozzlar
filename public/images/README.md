# Public Images Folder

This folder contains public image assets for the Wozzlar application.

## Required Files

- **wozzlar-logo.png**: High-resolution 512x512 PNG logo for the splash screen
  - Should be a clear, high-quality version of the Wozzlar logo
  - Will be displayed at 180x180px on the splash screen
  - Currently missing - please add this file

## Current Status

The splash screen currently uses `src/images/wozzlar-logo.png` which is a low-resolution icon file (48x48).
Please add the 512x512 PNG version to this folder and update the path in `src/index.html`.
